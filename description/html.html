HTML

Definisi :

HTML atau Hyper Text Markup Language terdiri dari dua istilah, yaitu : "Hypertext" dan "Markup Language". "Hypertext" adalah dokumen berisi tautan yang memungkinkan pengguna terhubung ke halaman lain. "Markup Language" merupakan bahasa komputer yang terdiri dari sekumpulan kode untuk mengatur struktur dan menyajikan informasi. Sehingga, HTML adalah bahasa markup standar yang digunakan untuk membuat struktur halaman web. HTML menggambarkan struktur sebuah halaman web yang terdiri dari serangkaian element. Dan elemen-elemen tersebut memberi tahu browser cara menampilkan konten. HTML dapat didukung dengan CSS maupun JavaScript.


Sejarah :

HTML dibuat oleh Sir Tim Berners-Lee, seorang ahli fisika di lembaga penelitian CERN yang berlokasi di Swiss. Versi pertama HTML dirilis pada akhir tahun 1991, tetapi tidak dirilis secara resmi, diterbitkan pada tahun 1995 sebagai HTML 2.0.
  • HTML 1.0 dirilis pada tahun 1993 dengan tujuan untuk berbagi informasi yang dapat dibaca dan diakses melalui browser web. Namun tidak banyak developer yang terlibat dalam pembuatan website. Jadi bahasanya juga tidak berkembang.
  • Kemudian muncullah HTML 2.0 , yang diterbitkan pada tahun 1995 , yang berisi semua fitur HTML 1.0 bersama dengan beberapa fitur tambahan itu, yang tetap sebagai bahasa markup standar untuk merancang dan membuat situs web hingga Januari 1997 dan menyempurnakan berbagai fitur inti HTML.
  • Kemudian muncullah HTML 3.0 , dimana Dave Raggett yang memperkenalkan makalah atau draft baru tentang HTML. Ini termasuk peningkatan fitur baru HTML, memberikan karakteristik yang lebih kuat untuk webmaster dalam merancang halaman web. Namun fitur canggih dari HTML baru ini memperlambat browser dalam menerapkan peningkatan lebih lanjut.
  • Kemudian datanglah HTML 4.01, yang banyak digunakan dan merupakan versi sukses dari HTML sebelum HTML 5.0 , yang saat ini dirilis dan digunakan di seluruh dunia. HTML 5 dapat dikatakan untuk versi perpanjangan dari HTML 4.01, yang diterbitkan pada tahun 2012 .
